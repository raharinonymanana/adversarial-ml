# Security / Model Assessment

This module differs from the attack-family folders. Rather than a single
attack class, it covers the standard tooling and methodology for assessing
the security of ML systems, as taught in NVIDIA DLI "Exploring Adversarial
Machine Learning" (Module 4):

- Adversarial Robustness Toolbox (ART) — attacking local models and remote endpoints
- TextAttack — generating adversarial text
- Alibi — explainability as a security/assessment tool
- Optuna — framing attacks as hyperparameter optimization

**Status:** lab in progress.
